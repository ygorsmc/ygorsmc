### 🚀 Featured Projects

Real projects, real metrics — each one deployed and documented.

#### 🏗️ [Daily Construction Report with Generative AI](https://ygorsmc.github.io/daily-construction-report-ai)
Serverless pipeline (**n8n**, 17 nodes) that generates an executive construction-progress report every workday — cross-referencing site logs with weather data through a **100% local LLM (Ollama / Qwen2.5)**, synthesizing audio and distributing via Telegram + e-mail with zero manual work.
> **-91% time** (~40 → ~3.7 min) · **R$ 0 operating cost** (fully local AI) · error handling & structured logging
> `n8n` · `Ollama / LLM` · `Python` · `REST APIs` · `TTS`

#### ✍️ [ENEM Essay Grader with AI](https://ygorsmc.github.io/enem-por-foto)
Multichannel bot (**WhatsApp + Telegram**) that grades Brazilian ENEM essays from a photo of the handwritten text: OCR transcribes it, a pluggable LLM scores it per the official INEP rubric with quote-grounded feedback.
> Accuracy **validated** (MAE 129 pts vs. the Essay-BR corpus, near-zero bias) · webhook **<200 ms** · **52 unit tests** · LGPD-compliant
> `FastAPI` · `Redis` · `Docker` · `Mistral OCR` · `LLMs` · `Azure`

#### 📊 [Customer Segmentation with RFM & K-Means](https://ygorsmc.github.io/RFM-Customer-Segmentation)
Unsupervised segmentation of e-commerce customers using the **RFM** methodology + **K-Means**, on real data structured in SQL, with an interactive client-side demo that returns a customer's segment in real time.
> K chosen via the **Elbow Method (K=4)** · 4 named business personas (Champions, At-Risk...) · real UCI/Kaggle data in SQLite
> `Python` · `Scikit-learn` · `SQL` · `K-Means` · `PyScript`

#### 🚢 [Titanic Survival Prediction](https://ygorsmc.github.io/Titanic-PyScript-Demo)
Classic classification pipeline (**Random Forest** & Logistic Regression) with EDA and a **100% client-side** deploy via PyScript — the model runs entirely in the browser, no back-end.
> **84% accuracy** · EDA (survival by sex, class, age) · missing-data imputation
> `Python` · `Scikit-learn` · `Pandas` · `EDA` · `PyScript`
